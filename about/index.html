关于本站

宝可梦冠军（Pokémon Champions）的对战数据库。汇总了游戏内排名画面的使用率数据，并做了中文本地化， 移动端同样友好。

数据来源

使用率数据镜像自 pokechamdb.com，并定期同步。招式元数据、种族值、可学习招式池等来自游戏解包数据。

本地增强

  • 全部文本中文化（宝可梦 / 招式 / 道具 / 特性 / 性格）
  • 属性弱点与抗性面板（4× / 2× / ½ / ¼ / 无效）
  • Mega / 地区形态切换，联动属性、种族值、特性、抗性
  • 招式面板附属性 / 物理特殊 / 威力 / 命中 / 先制值
  • 速度线对照（含 Mega 形态、与本地记录的我方队伍对比）

致谢

本站构建中使用或参考了以下开源项目：

  • naga0033/pokemon-champions-database — 上游项目，本站的数据模型与 UI 骨架 fork 自此
  • PokeAPI/pokeapi — 宝可梦种族值、属性、图鉴号数据
  • PokeAPI/sprites — 官方美术图 / HOME 精灵图
  • vercel/next.js — 全栈 React 框架（本站以静态导出模式构建）
  • tailwindlabs/tailwindcss — CSS 工具集
